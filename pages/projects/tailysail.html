TailySail Scoring System

Project Overview

TailySail has been my most complex project to date, The key has been to replace an aging excel workbork reliant on VBScript which lacks error handling (Typos can set off the results), and requires a large amount of data entry.

Requirements

  • Easy to use
  • Has Input Validation & Error Handling
  • Complies with the current sailing ruleset

Current Release

Currently, TailySail is a python program capable of Generating & Reading XLSX files, Interfaces with our club management system to parse attendee data, Processing either singular or multiple races, and generating results in PDF form.

Retrospective

  • Python is not suited for distributing to users.
  • Watch for scope creep.
  • UML Diagrams are superb for planning systems with a lot of attributes.

Future Versions

  • Migrating to C# & .NET
  • Watch for scope creep.
  • More planning via UML before writing classes.
