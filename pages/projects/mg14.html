MG14 Site

Project Overview

The MG14 Association was setting up a new Club Management system which included webhosting, I was tasked with creating the new site and organizing their domains.

Requirements

  • Simple for the executive comitee to edit.
  • Existing Honor Rolls needed to be preserved
  • Build with the limited CMS tools and WYSIWYG editor.

Ease of Editing

Pages were grouped in categories of "frequent" & "infrequent" edits. Frequent pages were designed to be easy to edit, whereas infrequent pages I used more advanced styling techniques

Honor Rolls

The Honor Rolls were 3 tables with approx 150 rows each. and the format needed to be adjusted. Rather than do this manually I wrote a python script to ingest the old rolls and transform them.

Current Status

The site has been handed over with the comitee doing minor updates, and leaving the occasional moderate updates to myself.
